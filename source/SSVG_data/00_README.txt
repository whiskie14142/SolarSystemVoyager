This folder contains a planetary ephemeris file "de440.bsp" at the installation of SSVG.

You can download the file "de440.bsp" from:
https://naif.jpl.nasa.gov/pub/naif/generic_kernels/spk/planets/de440.bsp
Related information can be retrieved from:
https://naif.jpl.nasa.gov/pub/naif/generic_kernels/spk/planets/
https://naif.jpl.nasa.gov/naif/

If you wish to use an old planetary ephemeris file "de430.bsp", download the file from 
https://naif.jpl.nasa.gov/pub/naif/generic_kernels/spk/planets/de430.bsp
