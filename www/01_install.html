SSVG はフリーソフトであり、どなたでも無料でお使いいただけます。
SSVG の動作は Windows 10（64ビット版）で確認しています。1280（横）×1024（縦）以上のサイズのディスプレイを持った PC が必要です。


（１）ダウンロード
Windows 用の SSVG のインストーラーはこちらからダウンロードできます。
SSVG_1_5_0_setup.exe


同じファイルは次のいずれかのサイトからダウンロードすることもできます。
Vector： SSVG 宇宙探査の旅
FreewareFiles： Solar System Voyager (SSVG)
Google ドライブ： SSVG-SolarSystemVoyager


（２）インストール
ダウンロードしたファイルをダブルクリックして実行すると、SSVG のインストールが始まります。
Windows が警告を表示して実行をブロックすることがありますが、実行の許可を与える操作を行ってください。
指示に従ってインストールを進めると5分ほどでインストールが完了します。

すでに SSVG を使用していた場合のデータファイルの取扱いについてはユーザーズガイドをご覧ください。


（３）SSVG の実行
インストーラーがデスクトップに作成した SSVG のショートカットをダブルクリックしてください。SSVG が起動します。Windows のスタートメニューから実行することもできます。

SSVG の使い方はユーザーズガイドをお読みください。ユーザーズガイドもデスクトップのショートカットで開くことができます。また、実行中の SSVG のメニューバーの「ヘルプ」から開くこともできます。

スマホで読む クイックリファレンス もどうぞ。→


（４）PythonのソースコードからSSVGを実行する
Python 3.8 の動く環境があれば、SSVG のソースコードをダウンロードして実行することもできます。ソースコードは次のサイトからダウンロードすることができます。
GitHub　https://github.com/whiskie14142/SolarSystemVoyager


インストール方法や詳細な使用法はユーザーズガイドをご覧ください。
ユーザーズガイドの最新版は次のリンクからダウンロードすることもできます。
日本語版 
英語版 


（５）旧型式の配布パッケージ（ZIP ファイル）
Windows 用の SSVG は、 ZIP ファイルをダウンロードしてインストールすることもできます。ダウンロードはこちらです。
SSVG_1_5_0.zip

  • ダウンロードしたファイルを Windows エクスプローラーなどの適当なツールで開いてください。
  • "SSVG_x_x_x" といった名前のフォルダがありますから、PC上の適当な場所にコピーしてください。
  • コピーしたフォルダを開いて SSVG の実行形式のプログラム（"SSVG.exe" または "SSVG" と表示されます）を見つけ、そのショートカットをデスクトップに作成してください。
  • ショートカットをダブルクリックして SSVG を実行してください。Windows が実行をブロックした場合は実行を許可する操作を行ってください（許可が必要なのは初回の実行だけです）。
  • これまでに作成した飛行計画があれば、そのファイルをすべてインポートしてください。インポートコマンドは SSVG の ファイル(F) メニューにあります。
  • これまでにダウンロードした小天体の SPK ファイルがあれば、そのファイルをすべてインポートしてください。
