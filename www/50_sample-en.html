This page provides additional samples of the Flight Plan of SSVG.

  • You need SSVG to run one of the Flight Plans. If you have not yet, please install it.
  • Choose one of the samples below.
  • Download the Flight Plan file and the SPK file of the sample.
  • Run SSVG and import the downloaded SPK file.
  • Import the downloaded Flight Plan file with SSVG.
  • Open the Flight Plan file with SSVG.
  • Read SSVG User's Guide to learn how to use SSVG.

 SFP001
Rendezvous with Ceres using EP

This Flight Plan makes the Probe rendezvous with dwarf planet Ceres. Ceres is the largest object in the main belt. The Probe uses its electric propulsion engine (EP) for the orbital transitions that make the Probe rendezvous with Ceres.

 SFP002
Rendezvous with Pallas using EP

This Flight Plan makes the Probe rendezvous with asteroid Pallas. Pallas is one of the largest four objects in the main belt. Since the orbital plane of Pallas inclines largely to the ecliptic plane, the Probe performs a swing-by of Jupiter in the first phase of the journey and approaches Pallas in the next phase.
In the Flight Plan, the Probe uses the electric propulsion engine to rendezvous with Pallas.
The Flight Plan is an improved version of "sample_Pallas" contained in SSVG.
