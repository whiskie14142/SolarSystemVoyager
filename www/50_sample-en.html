This page provides additional samples of the Flight Plan of SSVG.

  • You need SSVG to run one of the Flight Plans. If you have not yet, please install it.
  • Choose one of the samples below.
  • Download the Flight Plan file and the SPK file of the sample. Some samples may not contain any SPK file.
  • Run SSVG and import the downloaded SPK file if you have one.
  • Import the downloaded Flight Plan file with SSVG.
  • Open the Flight Plan file with SSVG.
  • Read SSVG User's Guide to learn how to use SSVG.

SFPA01
Rendezvous with Pallas

This Flight Plan makes the Probe rendezvous with the asteroid Pallas. It is one of the four large objects in the asteroid belt. Its orbital plane is inclined heavily from the ecliptic plane.
In the Flight Plan, the Probe flies to Jupiter, makes a swing-by, and then flies toward Pallas.

 SFPA01.json 2000002_Pallas.bsp
SFPA02
Spaceflight of Voyager 2

This Flight Plan replicates the spaceflight of Voyager 2. The spacecraft was launched in 1977 by the USA to probe Jupiter, Saturn, Uranus, and Neptune sequentially.

SFPA03
Flyby to Apollo

This Flight Plan makes the Probe flyby to Apollo. It is one of the asteroids that could come close to the Earth.

 SFPA03.json 2001862_Apollo.bsp
SFPA04
Rendezvous with Apollo

This Flight Plan makes the Probe rendezvous with Apollo. It is one of the asteroids that could come close to the Earth.
The Probe stays in the vicinity of Apollo for a long stint. The distance is about one hundred kilometers in the first phase and then comes closer to ten kilometers.
