Windows executable of SSVG was tested on Windows 8.1 (64bit) and Windows 10 (64bit). Recommended display size is 1024 pixels x 1280 pixels or more.


(1) Download
Installer of the Windows executable of SSVG is here.
SSVG_1_4_1_setup.exe 

You can download the same file from the following links:
Freeware Files: Solar System Voyager (SSVG)
Google Drive: SSVG-SolarSystemVoyager


(2) Install
Double click the downloaded file to start installation of SSVG.
When Windows blocks the execution of the installer, check the file is the correct one and give Windows permission to execute the file.
Go ahead with the installation under instructions from the installer. It takes about five minutes to complete.

If you have data files for older versions of SSVG, please check SSVG User's Guide.


(3) Run SSVG
Double click the shortcut of SSVG, which was created by the installer. A window of command prompt appeares, and SSVG starts the execution. You can run SSVG from the Start menu of Windows also.


(4) Run SSVG from Source Code
If you have an environment of Python 3.7, you can run SSVG from its source code. You can download source code of SSVG from the following link.
https://github.com/whiskie14142/SolarSystemVoyager

Read the document of SSVG to learn how to install the source code. You can download latest version of the documents from the following links.
English version 
Japanese version 


(5) ZIP file of Windows executable of SSVG (older format)
If you wish, you can download a ZIP file (compressed file) of Windows executable of SSVG and install SSVG from it. The file is here.
SSVG_1_4_1.zip

  • Open the downloaded file by your convenient tool (e.g. File Explorer)
  • Find a folder named something like "SSVG_x_x_x"
  • Copy the folder into your PC
  • Open the copied folder and find the executable program of SSVG. Its file name is "SSVG.exe" or "SSVG"
  • Create a shortcut of the file onto the Desktop of Windows
  • You do not need to download an SPK file named "de430.bsp" separately. The file is contained in the ZIP file
  • Run SSVG by double clicking of the shortcut of SSVG
  • If you have Flight Plan files of SSVG already, import all of them. You can find the Import command within the File menu of SSVG.
  • If you have SPK files for small celestial bodies, import all of them.
